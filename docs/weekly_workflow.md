# Weekly LinkedIn Hand-Raiser Workflow

## Monday Morning (15 minutes, 9am)

### **1. Review Last Week's Performance**
- [ ] Open Excel tracker: `tracking/linkedin_engagement_tracker.xlsx`
- [ ] Check Dashboard tab → Vertical Performance Summary
- [ ] Identify highest engagement vertical + pain point
- [ ] Note any posts with <2% engagement (kill those angles)

### **2. Plan This Week's Posts**
- [ ] Open post templates: `posts/post_templates.md`
- [ ] Select 3 posts (Mon/Wed/Fri):
  - **Monday:** Highest-performing pain point from last week
  - **Wednesday:** Second-highest performing pain point
  - **Friday:** New angle test (if bandwidth allows)
- [ ] Schedule in LinkedIn native scheduler (8am PST each day)

### **3. Pipeline Health Check**
- [ ] Check Response Tracker tab
- [ ] Count prospects awaiting video (Goal: <5)
- [ ] Count videos sent last week (Goal: 10+)
- [ ] Count demos booked (Goal: 2+)

---

## Daily Routine (10 minutes, 2x/day)

### **Morning Check (9am PST)**
- [ ] Monitor LinkedIn notifications
- [ ] Respond to comments on active posts (within 2 hours)
- [ ] Log new DMs in Response Tracker tab
- [ ] Send Loom videos to engaged prospects (use template from `/videos`)

### **Afternoon Check (3pm PST)**
- [ ] Update Post Performance Tracker with today's engagement data
- [ ] Follow up on any unanswered DMs
- [ ] Check for demo requests from video responders
- [ ] Update Response Tracker tab with video sends + demo bookings

---

## End of Week (Friday 4pm, 10 minutes)

### **Weekly Wrap-Up**
- [ ] Complete all Excel tracker entries for the week
- [ ] Screenshot top-performing post (save to `/posts` folder)
- [ ] Archive week's post copy + engagement data
- [ ] Review conversion funnel (Dashboard tab)
- [ ] Flag any stuck prospects (video sent >5 days, no demo booked)

---

## Monthly Review (1st Monday, 30 minutes)

### **Strategic Analysis**
- [ ] Review 4-week trend data (Dashboard tab → Weekly Trends section)
- [ ] Calculate cost per acquisition:
  - Time invested: __ hrs x $75/hr = $__
  - Deals closed: __
  - CPA: $__ ÷ __ = $__/customer
- [ ] Identify top 3 performing pain points (across all verticals)
- [ ] Identify bottom 3 (kill these angles)
- [ ] Update LinkedIn profile if positioning needs refinement
- [ ] Plan next month's test angles
